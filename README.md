# Bilingual Animal Learning Website

An interactive bilingual educational platform designed to help children learn animal names and pronunciations in both Indonesian and English through engaging digital content.

## Features

* 30+ interactive animal learning modules
* Bilingual content (Indonesian & English)
* Animal pronunciation and vocabulary learning
* Interactive quizzes for knowledge assessment
* Child-friendly and responsive user interface
* Technology-assisted learning experience for children, parents, and educators

## Purpose

This project aims to support early childhood language development by combining educational content with interactive web technologies. The platform encourages fun learning through visual, textual, and quiz-based activities while helping children improve vocabulary recognition in two languages.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Target Users

* Children
* Parents
* Teachers
* Early Childhood Education Institutions

## Key Highlights

* Supports bilingual learning in Indonesian and English
* Provides 30+ animal learning materials
* Enhances engagement through interactive quizzes
* Promotes technology-driven education and digital literacy
Website pembelajaran hewan dengan mode **Flashcard** dan **Quiz Mode**.

## Getting Started

1. Open `index.html` in your preferred web browser.
2. Select one of the available learning modes:
   - **📚 Flashcard Mode** – Explore animal names and pronunciations in Indonesian and English by flipping the cards and playing audio.
   - **🎯 Quiz Mode** – Test your knowledge by listening to animal sounds and choosing the correct answer.
