# Bilingual Animal Learning Website

Website pembelajaran hewan dengan mode **Flashcard** dan **Quiz Mode**.

## Cara pakai
1. Buka `index.html` di browser.
2. Pilih mode:
   - **📚 Flashcard**: klik kartu untuk membalik informasi dan tekan tombol **Sound**.
   - **🎯 Quiz Mode**: dengarkan suara lalu pilih jawaban yang benar.

## Stack
- HTML
- CSS
- JavaScript (vanilla)
